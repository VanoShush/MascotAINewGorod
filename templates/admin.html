⚙️ Настройки

Маскот будет работать только на этих сайтах. Вводите без протокола: noviy-gorod.ru

🔌 Интеграция с каталогом (опционально)

Позволяет маскоту искать товары/объекты через API вашего сайта. Если не заполнено — маскот работает без поиска по каталогу.

❓ Как заполнить эти поля?
Шаг 1 — Найдите URL поиска:
Откройте сайт → нажмите F12 → вкладка Network.
Сделайте поиск на сайте. Найдите запрос с JSON-ответом.
Скопируйте URL без параметров (всё до ?).

Шаг 2 — Опишите параметры:
- price_to: максимальная цена в рублях
- rooms: количество комнат (0 = студия)
- district: район на англ. (nevsky, central)
- area_min: минимальная площадь в кв.м
Шаг 3 — Опишите формат ответа:
[{"id":124,"title":"2к, 47м²","price":7800000,"url":"/flat/124"}]

📄 База знаний

Загрузите .txt или .md файл с описанием компании, услугами, ценами, правилами.
Маскот будет отвечать на вопросы на основе этой информации.
Максимум: 2 МБ. Можно загрузить несколько файлов.
